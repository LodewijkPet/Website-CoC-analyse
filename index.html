Code of conduct analyse

Deze website is gemaakt om code of conducts van bedrijven te analyseren. De code of conduct is een document waarin de regels en richtlijnen staan die worden gehanteerd. Deze website is gemaakt om de code of conduct te analyseren en te kijken welke regels en richtlijnen het meest voorkomen.
